Bookmarklet Dolibarr

Import en 1 clic depuis Dolibarr vers Gestion des Commandes

Le Bookmarklet

Glissez ce bouton dans votre barre de favoris

Exporter vers Gestion Commandes
Glissez-déposez ce bouton dans votre barre de favoris !

Installation

1
Affichez votre barre de favoris

Si elle n'est pas visible :

  • Chrome/Edge : Ctrl + Shift + B (Windows) ou Cmd + Shift + B (Mac)
  • Firefox : Ctrl + Shift + B (Windows) ou Cmd + Shift + B (Mac)
  • Safari : Cmd + Shift + B
2
Glissez le bouton violet dans vos favoris

Cliquez sur le bouton violet "Exporter vers Gestion Commandes" ci-dessus et glissez-le dans votre barre de favoris.

3
C'est terminé !

Le bookmarklet est maintenant installé. Vous pouvez l'utiliser sur n'importe quelle page de commande Dolibarr.

Utilisation

1
Ouvrez une commande dans Dolibarr

Allez sur la fiche d'une commande client dans votre Dolibarr

2
Cliquez sur le bookmarklet

Dans votre barre de favoris, cliquez sur "Exporter vers Gestion Commandes"

3
Le formulaire se remplit automatiquement !

L'onglet Gestion Commandes est réutilisé s'il est déjà ouvert — sinon un nouvel onglet est créé

Avantages du bookmarklet
  • Ultra rapide : 1 seul clic
  • Pas de copier-coller : Tout est automatique
  • Fonctionne toujours : Même si Dolibarr change
  • Sécurisé : Aucune donnée n'est envoyée à un serveur
Important
  • Vous devez être connecté à Dolibarr
  • Fonctionne uniquement sur les pages de commandes
  • Les popups doivent être autorisés pour ce site

Problèmes courants

Solutions :
  • Vérifiez que vous êtes bien sur une page de commande Dolibarr
  • Essayez de recharger la page Dolibarr
  • Vérifiez que les popups ne sont pas bloqués
  • Ouvrez la console (F12) pour voir les éventuelles erreurs

Possible si :
  • La page Dolibarr n'est pas complètement chargée
  • Votre version de Dolibarr a une structure HTML différente
  • Certains champs sont vides dans Dolibarr
Solution : Utilisez plutôt la méthode par URL (copier-coller l'URL)

Méthode alternative :
  1. Faites un clic droit sur le bouton violet
  2. Sélectionnez "Copier l'adresse du lien"
  3. Créez un nouveau favori manuellement
  4. Collez le lien copié comme URL du favori
